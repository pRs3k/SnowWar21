Bot .cfg files for Snow-War
===========================

Bots personalities are created using .cfg files, which are standard to Half-Life.  There are no formatting issues to deal with.  The user just sets the bot variables shown below in the example.cfg file.  You can rearrange or remove any of these variable without a problem.  Comments can be added any where on a line after the //.

// This is a comment.
//
// Bot name, model, and model colors
//
bot_name                 "Snow-Bot"
bot_model                "gordon"
bot_top_color            "255"
bot_bottom_color         "255"
//
// Bot traits
//
bot_trait_jump           52
bot_trait_accuracy       88
bot_trait_aggression     45
bot_trait_perception     75
bot_trait_reflexes       50
